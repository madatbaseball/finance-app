# 📊 Finance App

AI 기반 주가 분석 및 금융 정보 제공 웹 애플리케이션

## 주요 기능

- AI 기반 주가 분석 및 산업 리포트
- 환율 및 주요 경제 지표 모니터링
- 경제 캘린더 (주요 이벤트 일정)
- 포트폴리오 분석
- 종목 주가 검색

## 사용 기술

- **Frontend/Backend**: Streamlit
- **언어**: Python
- **API**: Yahoo Finance, KRX, Perplexity AI, DART(전자공시시스템)

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 제작 배경

비전공자가 금융 데이터를 직접 분석하고 공부하기 위해 제작한 개인 프로젝트입니다.
